# 🏋️ Iron Gym — Landing Page de Academia

Landing page moderna e estratégica desenvolvida para academias que buscam aumentar a captação de alunos através de uma presença digital forte, rápida e orientada à conversão.

## 🎯 Objetivo do Projeto

Criar uma landing page de alta conversão para o nicho fitness, focada em:

- Atrair novos alunos
- Gerar leads qualificados
- Destacar diferenciais da academia
- Melhorar a percepção de valor da marca

---

## ⚙️ Funcionalidades

- Layout moderno e responsivo (Mobile First)
- Seção hero com CTA estratégico
- Destaque de métricas (prova social)
- Estrutura otimizada para conversão
- Navegação suave (scroll behavior)
- Animações com Intersection Observer
- Design focado em performance

---

## 🧠 Estratégia aplicada

Este projeto não foi apenas visual — foi construído com base em princípios de:

- **UX (User Experience):** navegação clara e intuitiva  
- **UI (User Interface):** estética moderna e consistente  
- **Copywriting:** foco em ação e conversão  
- **Performance:** carregamento rápido e leve  

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox + Grid + Animações)
- JavaScript (Vanilla JS)
